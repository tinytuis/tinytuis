What it is

Single-page experiments. A word that unravels across the screen. Constellations that redraw with each keypress. A scrolling overheard chat. Each one short enough to feel disposable, sharp enough to leave an impression.

Caret Cuts is a collection of micro-programs—terminal haikus, digital sketches, computational doodles. Each experiment explores a single idea, executed quickly and discarded just as fast.

Run it

Terminal
# Clone to your home directory or a writable location
$ cd ~
$ git clone https://github.com/tinytuis/tinytuis.git
$ cd tinytuis
$ python issues/06-caret-cuts/main.py

Requires Python 3.7+ and the blessed library.

Notes

Inspired by the sketch pad, the notebook margin, the quick prototype. What happens when you treat code like a disposable medium? When programs become as ephemeral as thoughts?

Each experiment is designed to be written in under an hour, experienced in under a minute. They're meant to be rough, immediate, and surprising—like finding a crumpled note with a perfect line of poetry.

The collection includes text effects, generative patterns, interactive poems, data visualizations, and abstract animations. All rendered in the humble terminal, all designed to spark a moment of wonder.

Technical Details

Language: Python 3.7+
Dependencies: blessed, threading
Theme: Matrix green (terminal classic)
Input: Space, 'r' random, 's' save, 'q' to quit
