What it is

A dialogue with a machine that can only mirror and distort what you say. Type a word, watch it return to you bent, reversed, or scattered across the screen in different forms.

Each echo transforms your input through different filters—reversal, spacing, case changes, decay effects, and typographic distortions. The machine becomes a funhouse mirror for language.

Run it

Terminal
# Clone to your home directory or a writable location
$ cd ~
$ git clone https://github.com/tinytuis/tinytuis.git
$ cd tinytuis
$ python issues/04-echo-chamber/main.py

Requires Python 3.7+ and the blessed library.

Notes

Inspired by the way we talk to machines—and how they talk back. What happens when the only response is distortion? When communication becomes a game of telephone with yourself?

The program includes multiple echo modes: reverse, scatter, decay, whisper, shout, and stutter. Each mode applies different transformations to create a unique voice for the machine.

Try typing poetry, questions, or single words. Watch how different inputs create different patterns of distortion. The echo chamber becomes a collaborator in creating new forms of text.

Technical Details

Language: Python 3.7+
Dependencies: blessed, threading
Theme: Deep blue (terminal blues)
Input: Text input, tab to change modes, 'q' to quit
