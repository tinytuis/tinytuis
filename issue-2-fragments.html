What it is

A collage of text inside a black window. Half-finished thoughts, stray notes, and forgotten fragments drift by in an endless stream. You can catch them, let them pass, or stitch them together into something new.

Each fragment carries the weight of an incomplete thought—the grocery list that became poetry, the TODO that became philosophy, the random observation that became the seed of something larger.

Run it

Terminal
# Clone to your home directory or a writable location
$ cd ~
$ git clone https://github.com/tinytuis/tinytuis.git
$ cd tinytuis
$ python issues/02-fragments/main.py

Requires Python 3.7+ and the blessed library.

Notes

Inspired by the way we accumulate digital detritus—the notes app full of one-liners, the desktop scattered with text files, the browser tabs we never close. What if we could catch these fragments as they drift by?

The program reads from a collection of text fragments, but you can feed it your own notes, tweets, or any text file. Watch your own thoughts drift by, catch the ones that resonate, let the rest disappear into the digital ether.

Part text editor, part meditation on digital memory. A reminder that not everything needs to be finished to be meaningful.

Technical Details

Language: Python 3.7+
Dependencies: blessed, threading
Theme: Nebula (purples and blues)
Input: Arrow keys, space, enter, 'q' to quit
