Tiny TUIs · A study in beautiful computer-made text

Beautiful text for the command line age.

Tiny terminal interfaces, specimens, and patterns that treat text as craft. Variable weights, careful spacing, ligatures, and bitmap charm—applied to developer tools.

$ Text is the original interface.
$ Monospace is a design material.
$ Ligatures are tiny spells.
$ Bitmaps can be beautiful.

Computer-made text can be exquisite. Tiny TUIs explores the typographic surface of terminals—how spacing, weight, metrics, and rhythm shape the way developer tools feel. The project is split into experiments (little apps), patterns (reusable UI bits), and a small library that packages what we learn.

Typo note

The site uses system UI fonts and monospaced stacks by default, but each specimen exposes OpenType-like controls so you can feel how small adjustments change legibility and mood.

Principles

  1. Do the least, but do it well. Small programs. Great type.
  2. Make complexity invisible. Let rhythm and metrics carry the load.
  3. Actions first, details later. Text that moves you forward.
  4. Transparency should be useful. Show what matters; hide the rest.

Start with an experiment.

Clone a tiny app, tweak the text, feel the difference. Letterforms first, features second.
