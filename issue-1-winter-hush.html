What it is

Winter Hush invites you into a peaceful terminal snowfall. Move your cursor (represented by ^) left and right to catch falling snowflakes. Some snowflakes carry hidden text fragments—poetic whispers that reveal themselves when caught.

This is not a game with scores or levels. It's a meditative experience that explores the quietude of winter and the joy of discovering hidden meaning through gentle interaction.

Run it

Terminal
# Clone to your home directory or a writable location
$ cd ~
$ git clone https://github.com/tinytuis/tinytuis.git
$ cd tinytuis
$ python issues/01-winter-hush/main.py

Requires Python 3.7+ and the blessed library.

Notes

Inspired by the quiet beauty of winter and the way snow can make everything feel hushed and contemplative. The falling snowflakes are generated procedurally with simple physics—they drift slightly as they fall, making the animation feel more natural.

The text fragments are hidden in about 25% of the snowflakes. When you catch one, it gets added to your collection at the bottom of the screen. There's no goal or score—just the simple pleasure of catching fragments of poetry as they drift by.

The program uses threaded input handling so the snow keeps falling smoothly while you move the cursor. All the snowflakes and text are rendered in real-time using terminal escape codes.

Technical Details

Language: Python 3.7+
Dependencies: blessed, threading
Theme: Winter (ice blues and whites)
Input: Arrow keys, 'q' to quit
